Conjuntos

Definições e operações fundamentais

Um conjunto é uma coleção de objetos distintos, chamados elementos. Os elementos podem ser números, letras, ou qualquer objeto definido.

Exemplo: A = {1, 2, 3, 4, 5}

Notação: 3 ∈ A (3 pertence a A)

A união de dois conjuntos A e B é o conjunto formado por todos os elementos que pertencem a A ou a B (ou a ambos).

A = {1, 2, 3}

B = {3, 4, 5}

A ∪ B = {1, 2, 3, 4, 5}

A interseção de dois conjuntos A e B é o conjunto formado pelos elementos que pertencem simultaneamente a A e a B.

A = {1, 2, 3}

B = {3, 4, 5}

A ∩ B = {3}

O complemento de um conjunto A em relação a um conjunto universo U é o conjunto de elementos que pertencem a U mas não pertencem a A.

U = {1, 2, 3, 4, 5}

A = {1, 2}

A' = {3, 4, 5}

Um conjunto A é subconjunto de B (A ⊆ B) se todos os elementos de A também pertencem a B.

A = {1, 2}

B = {1, 2, 3, 4}

A ⊆ B (A é subconjunto de B)

Relações

Propriedades de relações entre elementos

Uma relação R é reflexiva se todo elemento se relaciona consigo mesmo. Para todo a ∈ A, temos (a, a) ∈ R.

Exemplo: "é igual a"

5 = 5 (todo número é igual a si mesmo)

Uma relação R é simétrica se quando a se relaciona com b, então b também se relaciona com a. Se (a, b) ∈ R, então (b, a) ∈ R.

Exemplo: "é irmão de"

Se João é irmão de Maria, então Maria é irmã de João

Uma relação R é transitiva se quando a se relaciona com b e b se relaciona com c, então a se relaciona com c. Se (a, b) ∈ R e (b, c) ∈ R, então (a, c) ∈ R.

Exemplo: "é menor que"

Se 2 < 5 e 5 < 10, então 2 < 10

Uma relação é de equivalência quando é reflexiva, simétrica e transitiva simultaneamente.

Exemplo: "tem a mesma idade que"

• Reflexiva: você tem a mesma idade que você

• Simétrica: se A tem a mesma idade que B, B tem a mesma idade que A

• Transitiva: se A tem a mesma idade que B e B tem a mesma idade que C, então A tem a mesma idade que C

Funções

Tipos e propriedades de funções

Uma função f: A → B é uma relação que associa cada elemento do conjunto A (domínio) a um único elemento do conjunto B (contradomínio).

f(x) = 2x + 1

Se x = 3, então f(3) = 2(3) + 1 = 7

Uma função é injetora quando elementos diferentes do domínio têm imagens diferentes. Se f(a) = f(b), então a = b.

f(x) = 2x é injetora

Valores diferentes de x sempre resultam em valores diferentes de f(x)

Uma função é sobrejetora quando todo elemento do contradomínio é imagem de algum elemento do domínio. A imagem é igual ao contradomínio.

f: ℝ → ℝ, f(x) = x + 1

Para qualquer y ∈ ℝ, existe x = y - 1 tal que f(x) = y

Uma função é bijetora quando é simultaneamente injetora e sobrejetora. Existe uma correspondência um-a-um entre domínio e contradomínio.

f: ℝ → ℝ, f(x) = 3x - 2

É injetora e sobrejetora, portanto bijetora

A composição de funções f e g, denotada por (f ∘ g)(x) = f(g(x)), é aplicar primeiro g e depois f ao resultado.

f(x) = 2x, g(x) = x + 3

(f ∘ g)(x) = f(g(x)) = f(x + 3) = 2(x + 3) = 2x + 6

A função inversa f⁻¹ existe se f é bijetora. Ela "desfaz" a operação de f: se f(a) = b, então f⁻¹(b) = a.

f(x) = 2x + 1

f⁻¹(x) = (x - 1) / 2

Verificação: f(f⁻¹(x)) = x e f⁻¹(f(x)) = x

Sequências Numéricas

Progressões aritméticas e geométricas

Uma sequência é uma PA quando a diferença entre termos consecutivos é constante (razão r).

Termo Geral:

aₙ = a₁ + (n - 1) · r

Exemplo:

PA: 2, 5, 8, 11, 14, ... (r = 3)

a₅ = 2 + (5 - 1) · 3 = 14

Soma dos n primeiros termos:

Sₙ = n(a₁ + aₙ) / 2

Uma sequência é uma PG quando a razão entre termos consecutivos é constante (razão q).

Termo Geral:

aₙ = a₁ · q⁽ⁿ⁻¹⁾

Exemplo:

PG: 2, 6, 18, 54, ... (q = 3)

a₄ = 2 · 3³ = 54

Soma dos n primeiros termos:

Sₙ = a₁(qⁿ - 1) / (q - 1), para q ≠ 1

Soma infinita (|q| < 1):

S∞ = a₁ / (1 - q)
